網頁瀏覽縮放

  • Ctrl + 滑鼠滾輪向上：放大
  • Ctrl + + / -：調整倍率
  • Ctrl + 0：還原 100%

流動版面

固定版面

放大後超出頁面，需調整橫向捲軸

電腦、手機與 AI

作業系統調整（Win + U）

  • 字體、圖示放大
  • 顏色、對比
  • 編輯游標、滑鼠指標放大
  • 醒目提示
  • 其他輔助設定

輕鬆存取/協助工具細項

  • 文字大小　顯示比例　螢幕解析度
  • 滑鼠指標　文字游標　色彩濾鏡

開啟方式

Windows 10：Win + U → 輕鬆存取

Windows 11：Win + U → 協助工具

電腦、手機與 AI

NVDA　NonVisual Desktop Access

  • 開放原始碼自由軟體，任何人可使用、複製、散佈、研究、改良
  • 有眾多附加元件（add-on），可依需求選用功能
  • 語音引擎具有彈性，可選用多種語音
  • 支援多數點字顯示器，可同步點字輸出
  • 有豐富的網路資源與社群可尋求協助
