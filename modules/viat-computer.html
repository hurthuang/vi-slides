電腦視覺輔助設定

網頁瀏覽縮放

  • Ctrl + 滑鼠滾輪向上
  • Ctrl + + / - 調整倍率
  • Ctrl + 0 還原 100%

注意版型差異：流動版面自動斷行（Gmail/Facebook）；固定版面放大後需捲動橫軸。

作業系統調整（Win + U）

  • 字體、圖示放大
  • 顏色、高對比模式
  • 編輯游標、滑鼠指標放大
  • 醒目提示
  • 色彩濾鏡（色弱輔助）

Windows 10 輕鬆存取 ／ Windows 11 協助工具（Win + U）

NVDA　NonVisual Desktop Access

  • 開放原始碼自由軟體，任何人可使用、複製、散佈、研究、改良
  • 有眾多附加元件（add-on），可依需求選用功能
  • 語音引擎具有彈性，可選用多種語音
  • 支援多數點字顯示器，可同步點字輸出
  • 有豐富的網路資源與社群可尋求協助
