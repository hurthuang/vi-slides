格式調整：文字部份

  • 版面順序：有分欄、圖文框時，需確認版面順序是否正確
  • 避免純視覺格式（粗體、顏色）作為唯一區分
  • 保留語意結構（標題層級、清單）

格式調整：表格部份

改為純文字敘述

先報完整列，再報下一列；或先報完整欄，再報下一欄

保留表格

表格空間概念較佳者可利用 NVDA：
Ctrl + Alt + 方向鍵 跨欄列跳轉

格式調整：圖形部份

低視力生

  • 電子檔搭配電腦功能放大
  • 紙本搭配光學輔具放大

重度視障生

  • 依圖形特性轉化描述
  • 立體圖 / 觸覺圖輔助

格式調整：數理符號

Access8Math（A8M）

數學物件可輸出視覺數學式、語音、點字

MathType

通用數學物件編輯，搭 A8M 使用

相關筆記： 淺談重度視障者的數學書寫方式 ↗ 數學點字（Nemeth）↗

格式調整工具

  • 鄭明芳老師「中明試卷製作 ↗」Word 巨集（使用前請先在 C 槽新增資料夾 temp）
  • 試卷全自動調整：Alt+F9（一般試卷適用）
  • 細節批次調整：表格整理（橫式 ／ 直式 ／ 純文字還原）

其他常用功能

  • Ctrl + B：手動六點輸入點字
  • 批次替換數學符號
  • 段落間距、字距調整

主要刪改原則：文字部份

字形字音（含諧音）題目，使用語音報讀或點字呈現時，可能洩漏答案或無法擷取題意關鍵：

  • 仍可依賴視覺讀題：直接參閱紙本試卷
  • 無法以視覺讀題：刪除該題，重新配分

主要刪改原則：圖表部份

  • 仍可依賴視覺讀題：直接參閱紙本試卷
  • 圖形有文字替代描述者：可保留並提供描述
  • 無法轉化的純圖形題：酌予刪題重新配分
