UEB 縮寫類型

  • Shortform　字母組合代表單字（如 ab = about）
  • Alphabetic Wordsigns　單一字母代表整個單字（如 b = but）
  • Strong Contraction　任何位置都要優先縮寫（and / for / of / the / with）
  • Strong Groupsigns/Wordings　可當字根，也可獨立成單字（如 ch = child）
  • Lower Groupsigns/Wordings　下位字根，在字首、字中、獨立時意義不同（如 ⠆ = be / bb）
  • Initial-Letter Contractions　前置符號 + 啟始字母代表單字（如 ⠐⠎ = some）
  • Final-Letter Groupsigns　前置符號 + 結尾字母作為字根，不可獨立成字（如 ⠨⠎ = less）
