NVDA 焦點

焦點種類

  • 鍵盤焦點
  • 滑鼠焦點

特殊情況

九宮格（NumLock 關）：
0 + * → 焦點移至滑鼠指標處

使用 NVDA 瀏覽網頁

基本瀏覽

  • 方向鍵上下：逐行瀏覽
  • Tab：跳到下一個連結／按鈕
  • Enter：開啟連結

切換模式

  • 瀏覽模式 ↔ 焦點模式
  • NVDA + Space

單鍵瀏覽

鍵 跳轉目標（加 Shift 往回）
H標題
G圖片
B按鈕
K連結
E編輯欄
T表格

瀏覽網站

  • 初次瀏覽網站：全部跳轉過一次
  • 記錄有哪些物件，以及物件的類型
  • 下次瀏覽就能快速跳到要操作的物件

YouTube 範例

  1. Alt + D → 網址列，輸入 youtube.com
  2. E → 進入搜尋編輯區
  3. Enter → 進入焦點模式，輸入關鍵字
  4. Enter 送出搜尋
