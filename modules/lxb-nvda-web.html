NVDA 焦點

焦點種類

  • 鍵盤焦點
  • 滑鼠焦點
  • NVDA 焦點

焦點切換

  • NVDA + 6：檢閱游標跟隨編輯游標（建議開）
  • NVDA + 7：導航器物件跟隨系統焦點（建議開）
  • NVDA + 8：自動將系統焦點移至可聚焦項目（建議開）
  • NVDA + M：跟隨鼠標朗讀（依需求設定）
  • Ctrl + NVDA + T：點字跟隨（建議自動）

NVDA 焦點

特殊情況（九宮格，NumLock 關的模式下）

0 + *（星號）焦點移到滑鼠指標所在處（筆電：NVDA + Shift + M）
0 + /（斜線）滑鼠指標移到焦點所在處（筆電：NVDA + Shift + N）
0 + −（減號）焦點移到系統焦點所在處

使用 NVDA 瀏覽網頁

基本的網頁瀏覽：

  • 方向鍵上下：網頁物件跳轉
  • Tab ／ Shift + Tab：跳轉至下一個 ／ 上一個可互動項目
  • 空白鍵：選取
  • Enter：執行

瀏覽網站

  • 初次瀏覽網站：全部跳轉過一次
  • 記錄有哪些物件，以及物件的類型
  • 下次瀏覽就能快速跳到要操作的物件

YouTube 範例

  1. Alt + D → 網址列，輸入 youtube.com
  2. E → 進入搜尋編輯區
  3. Enter → 進入焦點模式，輸入關鍵字
  4. Enter 送出搜尋
