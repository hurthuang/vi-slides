⌨ 六點輸入法 (6d-IME)

Ctrl + NVDA + 7開啟／關閉六點輸入
J + K + L + Space循環切換點字模式
Ctrl + NVDA + 8模式偏好設定
Alt + Shift + NVDA + MNemeth 數學符號選單

支援的點字模式

  1. 注音點字
  2. 黑點點字（可自訂）
  3. 電腦點字（Comp8）
  4. UEB 一級
  5. UEB 二級

可在偏好設定中勾選要參與循環切換的模式
